# EVLite - PremierInfo
**Motor de comandos personalizados infinito basado en YAML para PocketMine-MP.**

Este plugin permite crear comandos con subcomandos ilimitados, mensajes personalizados y sonidos, todo desde la configuración.

### ✨ Características
* **Comandos Dinámicos:** Crea tantos subcomandos como necesites (ej: `/ayuda 1`, `/ayuda 2`).
* **Placeholders:** Soporte para `{player}`, `{online}`, `{command}` y `{subcommand}`.
* **Sistema de Sonidos:** Feedback auditivo (positivo, info, negativo).

### 🛠️ Estructura de archivos
Para que el plugin funcione, asegúrate de subir los archivos así:
- `plugin.yml` -> Raíz del proyecto.
- `src/EVLitePremierInfo/` -> Archivos .php.
- `resources/config.yml` -> Tu configuración de comandos.
